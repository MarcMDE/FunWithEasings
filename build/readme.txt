CONTROLS
-------------------------------------
Arrow Keys (Right/Left) -> Switch Easing "Type".
Arrow Keys (Up/Down) -> Switch Easing "Mode".
Space -> Play Easing.
A/D -> Increase/Decrease Easing Duration.
G -> Ghosts On/Off.
C -> Random colors. Bicos yes.

R -> Some stupid fun.
